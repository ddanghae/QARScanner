스캔 모드

현재 모드 위 필터 바의 "스캔 모드"에서 변경

급락 반등은 크게 떨어진 코인이 되돌아오는 자리를 찾습니다. 조기 포착은 반대로, 아직 조용히 바닥에서 변동성이 눌려 있고 거래량이 마른 상태에서 미결제약정이 늘고 있는 코인을 찾아 매집 → 임박 → 돌파 3단계로 보여줍니다. 크게 오르기 전을 노릴 때 조기 포착을 쓰세요. 단, 언제 오를지는 알 수 없고 오르지 않을 수도 있습니다.

채점 강도

채점을 얼마나 깐깐하게 할지 정합니다. 숫자가 낮을수록 조건을 느슨하게 봐서 더 많은 코인이 목록에 나오고, 높을수록 확실한 것만 남겨 적게 나옵니다. 스캔 결과가 너무 적으면 숫자를 내리세요.

특별 조건 켜기 / 끄기

4시간봉에서 50일선이 200일선을 위로 뚫은 뒤, 되돌림에서 그 선을 다시 밟고 위로 찔렀다 꺾이는 거부 캔들이 나온 자리만 보여줍니다. 상승 추세 초입을 노릴 때 켜세요. (조건이 까다로워 매우 적게 나옵니다.)

1시간봉 가격이 200일선에 딱 붙어있는 코인만 보여줍니다. 200일선에서 튕기는 지지·저항 반응을 노릴 때 켜세요.

위 "200선 밀착"을 얼마나 넓게 볼지입니다(ATR 배수). 숫자가 크면 조금 멀어도 밀착으로 봐서 코인이 많아지고, 작으면 아주 딱 붙은 것만 봅니다. 기본 0.5.

이미 많이 올라 진입하기 늦은 5단계 코인을 목록에서 뺍니다. 뒤늦게 따라 사는 것을 막고 싶을 때 켜세요.

상장한 지 얼마 안 돼 과거 데이터가 부족한 코인을 뺍니다. 계산이 불안정한 신규 코인을 피할 때 켜세요.

방향 없이 톱니처럼 오르내리는 횡보 구간이나 거래가 거의 없는 코인의 신호를 뺍니다. 후보 조건을 넉넉하게 잡는 대신, 되돌림에 쉽게 뒤집히는 믿기 어려운 신호를 제거하는 장치입니다. 켜두는 걸 권장합니다(기본 켜짐).

계산 방식 · 갱신

켜면 아직 마감 안 된 캔들도 계산에 넣어 신호가 더 빠릅니다. 대신 캔들이 마감되며 신호가 바뀔 수 있습니다(리페인트). 안정적으로 보려면 꺼두세요(기본 꺼짐).

일정 시간마다 자동으로 다시 스캔합니다. 켜두면 계속 최신 후보를 받아볼 수 있습니다.

몇 초마다 자동으로 다시 스캔할지입니다. 짧을수록 자주 갱신되지만 그만큼 API 요청이 많아집니다. 너무 짧으면 30초로 자동 제한됩니다.
